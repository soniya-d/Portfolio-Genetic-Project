Genetic Optimization Framework with Diverse Operators in Python

  • Built a Python-based Genetic Algorithm that can solve both function optimization and pathfinding (like TSP) problems.
  • Integrated different crossover and mutation methods (one-point, order, Gaussian, and swap) that can be swapped and tested.
  • Ran tests on benchmark problems (Sphere, Rosenbrock, Himmelblau, TSP) to compare how each operator performs.
  • Visualized algorithm behavior using matplotlib to analyze fitness trends, convergence, and computation time.
